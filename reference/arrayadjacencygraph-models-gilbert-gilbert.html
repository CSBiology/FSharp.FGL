gilbert numberOfVertices probability isDirected fVertexKey fLabel fWeight

Full Usage: gilbert numberOfVertices probability isDirected fVertexKey fLabel fWeight

Parameters:
    numberOfVertices : int
    probability : float
    isDirected : bool
    fVertexKey : int -> 'Vertex
    fLabel : 'Vertex -> 'Label
    fWeight : 'Vertex * 'Vertex -> 'Edge

Returns: ArrayAdjacencyGraph<'Vertex, 'Label, 'Edge>

Returns an ArrayAdjacencyGraph that is generated randomly with the given parameters. numberOfVertices indicates the number of vertices the final graph will have. probability represents the probability of an edge between 2 vertices. fVertexKey is a function that is used to transform an integer (the index of the vertex) into the 'Vertex type. fLabel is a function that transforms the 'Vertex type into a label of the 'Label type. fWeight is a funtion that takes two 'Vertices and returns a weight between them in form of an 'Edge type.
