louvain graph modularityIncreaseThreshold

Full Usage: louvain graph modularityIncreaseThreshold

Parameters:
    graph : ArrayAdjacencyGraph<'Vertex, 'Label, float>
    modularityIncreaseThreshold : float

Returns: ArrayAdjacencyGraph<'Vertex, ('Label * int), float>

Takes an ArrayAdjacencyGraph and returns a new graph whose Labels have been transformed into tupels, where the second part is the community accorging to modularity-optimization. Parameters: graph : ArrayAdjacencyGraph, that is used as the template for the modularity optimization. modularityIncreaseThreshold : Threshold of modularity-difference that has to be exceeded in order to be recognized as a modularity-increase. The value has to be between 0. and 1. to get a meaningful result. The smaller the value, the longer the calculation takes.
