Returns: ArrayAdjacencyGraph<'Vertex, 'Label, 'Edge>

Returns an ArrayAdjacencyGraph that was randomly grown according to the Barabási–Albert model with the given parameters. startingGraph is the original graph, that is used as the initial connected network. The rest of the calculations and growth of the network are performed on this graph. numberOfVertices specifies how many additional vertices the final graph will have. numberOfEdgesPerIteration specifies how many edges should be added to the graph per iteration. fVertexKey is a function that is used to transform an integer (the index of the vertex) into the 'Vertex type. fLabel is a function that transforms the 'Vertex type into a label of the 'Label type. fWeight is a funtion that takes two 'Vertices and returns a weight between them in form of an 'Edge type.
